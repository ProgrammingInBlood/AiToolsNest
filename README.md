## Tech Stack

**Client:** Nest.js, Zustand, TailwindCSS

**Server:** Node, Express

## Run Locally

Clone the project

```bash
  git clone https://github.com/ProgrammingInBlood/AiToolsNest
```

Go to the project directory

```bash
  cd AiToolsNest
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
